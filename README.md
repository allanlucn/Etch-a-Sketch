Etch-a-Sketch (Lousa mágica)
===========================

Aplicação simples de desenho em grade feita com HTML, CSS e JavaScript.

[Acesse](https://allanlucn.github.io/Etch-a-Sketch/)

O que o programa faz
--------------------
- Permite desenhar em uma grade clicando e arrastando.
- Possui seletor de cor para escolher a cor do desenho.
- Modo "Cores Aleatórias" (arco-íris) para desenhar com cores variadas.
- Modo borracha para apagar células individuais.
- Botão para limpar toda a grade.
- Botão para ajustar o tamanho da grade (entrada do usuário, máximo 100).
- Ícones no cabeçalho com links para LinkedIn e GitHub.

Arquivos principais
-------------------
- `index.html` — estrutura e ícones.
- `styles.css` — estilos e layout.
- `draw.js` — lógica de criação da grade e interação de desenho.
